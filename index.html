This page has a tiny amount of CSS applied by default (in theme.css), so it is pretty close to the appearance of raw HTML. Try resizing your browser and appreciate that it adapts to different screen sizes quite nicely right out of the box!

Edit the style.css file to complete the objectives described below. You will need to look at index.html to find the appropriate element names and classes, and then target them with CSS Selectors to apply specific properties. Links are included to guide you, but if you are struggling, review the articles at Interneting is Hard or contact your instructor.

Note: The checkboxes will not persist when the page refreshes, but may be used to double-check your work or during instructor feedback.

Challenges

  1. Give a max-width to the <article> element in order to constrain its line length to a resonable width. Then use margins to center the <article> element horizontally within the page. It should look like a typical article or blog post with space on the left and right. To visualize the <article> element on the page, you can temporarily give it a background-color or border.
  2. Change the color of all hyperlinks during a mouse hover event.
  3. Apply a new background-color to the <nav> element and add some padding.
  4. Remove the bullet points on all the nav links with list-style-type
  5. Make the list go horizontally by applying display: flex to to the unordered list, then space the items out equally with justify-content. Read more about Flexbox if needed.
  6. Reverse the order of the nav items with flex-direction.
  7. Give a unique text or background color for only the nav item reading "Nowhere". (You could use a class name or a pseudo-selector to do this)
  8. Use a fluid unit, such as vw to have the font-size of the <h1> change with the browser's width. You can set a minimum and maxium font-size by using clamp. Resize your window (or the Glitch Preview pane) to test.
  9. Give the header (the "hero" area containing the page title) a substantial height (at least 300px) and a linear-gradient background-image.
  10. Finally, center the title horizontally and vertically within the header using Flexbox. Once again, if the text color is hard to make out, you can change it's color to ensure sufficient contrast.
